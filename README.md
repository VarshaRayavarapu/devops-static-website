# DevOps CI/CD Static Website
![Deploy Website](https://github.com/VarshaRayavarapu/devops-static-website/actions/workflows/deploy.yml/badge.svg)

This project demonstrates a simple DevOps CI/CD pipeline using GitHub Actions and GitHub Pages.

## Objective
The objective of this project is to create and deploy a static website using an automated CI/CD pipeline.

## Tools Used
- VS Code
- Git
- GitHub
- GitHub Actions
- GitHub Pages

## Workflow
1. Develop website files in VS Code
2. Track and manage code using Git
3. Push code to GitHub repository
4. Trigger GitHub Actions workflow automatically
5. Deploy website to GitHub Pages

## Live Website
[View the deployed website](https://varsharayavarapu.github.io/devops-static-website)

## Outcome
This project demonstrates DevOps automation, continuous integration, and continuous deployment.
